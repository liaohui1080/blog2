# 1. Teleport 报错

# 错误原因

在 Nuxt 3 的 SSR 场景中，建议包裹在 <ClientOnly> 里,否则会出现找不到渲染的 teleport的节点的问题
1
2

# 错误代码

大概就是找不到渲染的 teleport的节点

# 解决方案

<!--在组件中使用 Teleport-->
<ClientOnly>
  <Teleport to="#portal">
    <!-- Teleport 内容 -->
  </Teleport>
</ClientOnly>


<!-- app.vue -->
<!-- 用作 Teleport 的目标 -->
  <div id="portal"></div>
